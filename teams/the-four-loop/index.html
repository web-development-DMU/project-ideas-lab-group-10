Application

SourceFlow — Personal Shopping Request System

What we are building

SourceFlow is a web-based personal shopping request manager. Customers submit sourcing requests, track progress, and update their details. The concierge team reviews requests, updates statuses, and adds notes.

The goal is a simple database-driven application that demonstrates CRUD operations using SQLite, delivered through a JavaScript web server and semantic HTML/CSS pages.
